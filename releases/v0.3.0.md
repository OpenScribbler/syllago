### New Features

#### CLI Commands
- **`syllago export`**: Export items from `local/` to a provider's install location. Supports `--to <provider>`, `--type`, and `--name` flags. Skips JSON-merge types (MCP, Hooks) with a warning.
- **`--name` flag for `syllago import`**: Case-insensitive substring filter to import specific items instead of scanning everything. Composable with `--type`.
- **`--yes`/`-y` flag for `syllago init`**: Skip confirmation prompts for scripting. Also auto-detects non-interactive shells (no TTY) and skips prompts automatically.

#### TUI
- **Responsive welcome page**: Three layout tiers based on terminal height â€” ASCII art + cards (>= 42), cards only (>= 35), compact text list (< 35).

### Improvements

#### CLI
- **Add success message**: Now shows next-steps hint pointing to `syllago` TUI and `syllago export`.
- **Discovery diagnostics**: When import finds nothing, shows whether the content type is unsupported or which paths were searched and came up empty.
- **Skills discovery**: `findFiles()` now recurses one level into subdirectories, fixing discovery of directory-structured content like Claude Code skills.

#### TUI
- **Sidebar stability**: Version line hides cleanly when terminal is too short instead of causing erratic bottom behavior.
- **Detail scroll bounds**: Scroll offset is now floored at 0, preventing negative index panics.

### Bug Fixes

- Fixed panic (`slice bounds out of range [-1:]`) when scrolling up past the top of a detail page.
- Fixed skills not being discovered for claude-code provider when stored as directories.
- Fixed `syllago init` hanging in non-interactive shells waiting for confirmation prompt.
- Fixed sidebar bottom jumping erratically when resizing terminal vertically.
- Fixed welcome page cards being truncated at medium terminal heights.

### Breaking Changes

- **Removed experimental commands**: `syllago scan`, `syllago baseline`, `syllago drift`, `syllago parity` have been removed. These were pre-v1 experiments that are no longer part of the product direction.
- **Removed internal packages**: `scan/detectors`, `drift`, `parity`, `emit`, `reconcile` packages deleted.
- **Removed `.romanesco.yaml` files**: Replaced by `.syllago.yaml` as part of the rename.
