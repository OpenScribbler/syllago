## v0.4.1

### Highlights

**Release Skill** — `/release <version>` automates the full release workflow: version bump, release notes, commit, tag, push, and GitHub release creation.

---

### New Features

- **`/release` project skill**: Claude Code skill that orchestrates the entire syllago release process. Validates semver ordering, gathers change context from `git log` and `git diff --stat`, writes release notes from the template, bumps `VERSION`, commits, tags, pushes, and creates a GitHub release via `gh`.
