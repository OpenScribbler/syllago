description: Complete implementations — no TODOs or placeholder code
alwaysApply: true
---

Never write placeholder implementations. Incomplete code ships as bugs.

Do not write:
- `// TODO: implement this`
- Empty function bodies where logic is expected
- Stub returns that silently succeed or return zero values
- Partial logic with `// rest of this later`

If a full implementation is not possible in the current context, say so explicitly rather than leaving code that appears to work but does not.
