# Concise Output

A rule that instructs Claude Code to keep responses focused and avoid unnecessary verbosity.

## What it does

Sets communication guidelines for Claude Code:
- Avoids verbose preambles and summaries
- Keeps responses focused on the task
- Only provides detail when explaining trade-offs or when explicitly asked

## When to use

Install this rule when you want Claude Code to be more direct and skip filler text. Especially useful for experienced users who don't need step-by-step narration.
