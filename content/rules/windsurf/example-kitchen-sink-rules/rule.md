# Kitchen Sink Windsurf Rule

This is an example rule that demonstrates the always-active mode for Windsurf rules. It applies unconditionally to every conversation.

## Guidelines

These guidelines apply to all interactions:

1. **Be concise** - Lead with the answer, skip unnecessary preamble
2. **Show don't tell** - Provide code examples instead of lengthy descriptions
3. **Cite sources** - When referencing documentation, include the relevant link
4. **Structured output** - Use headings, lists, and code blocks for readability
