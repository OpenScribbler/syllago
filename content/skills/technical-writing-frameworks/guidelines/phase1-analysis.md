# Phase 1: Analysis

Analyze persona feedback to prepare for synthesis. Do NOT present a plan yet.

## Analysis Checklist

### 1. Pattern Recognition
Identify recurring themes across 3+ personas. Document each pattern with: who mentioned it, severity per persona, common request.

### 2. Conflict Detection
Find contradictory feedback (one persona wants MORE detail, another wants LESS). Document: who wants what, evidence from feedback, nature of conflict (usually audience mismatch).

### 3. Severity Assessment

| Level | Criteria |
|-------|----------|
| **Critical (P0)** | Blocks task completion, missing essential info, dangerous/incorrect content, multiple personas flag Critical |
| **Major (P1)** | Significantly degrades experience, important content missing but workarounds exist, multiple personas flag Major |
| **Minor (P2)** | Quality improvements, enhancement requests, single Major from one persona |
| **Suggestions (P3)** | Nice-to-have, future considerations, single persona suggestions |

### 4. Access Method Analysis
How users arrived: search with specific intent, links from other docs, incident/urgent situations. Flag mismatches between arrival method and document content.

### 5. Experience Level Analysis
Are beginners in advanced content? Experts frustrated by hand-holding? Flag mismatches between content level and user expertise.

### 6. Framework Pre-Assessment
Start thinking about: Which 7-Action actions are personas trying to accomplish? Which does the document address? What Diataxis type is this? What type are users expecting?

### 7. Preliminary Prioritization
Create rough buckets: Must Fix (P0), Should Fix (P1), Consider (P2), Future (P3).

## Completion Checkpoint

Before Phase 2, confirm:
- 2-5 major cross-persona patterns identified
- Conflicts documented
- Severity assessed for all items
- Access method and experience level mismatches noted
- Preliminary priority buckets created

## Output Format

```
Analysis complete. Ready for synthesis phase.

Key findings:
- [2-3 sentence summary of major patterns]
- [1-2 sentence note on conflicts]
- [1 sentence on severity distribution]
```
