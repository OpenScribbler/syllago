# Agent Skills Spec: Community Sentiment Research

**Date:** 2026-04-01
**Scope:** Deep research across GitHub, Reddit, HN, blogs, Twitter/X, and community channels
**Subject:** https://agentskills.io — the open Agent Skills specification

## Research Summary

The Agent Skills spec has achieved remarkable adoption (33+ tools) through radical simplicity — a folder with a markdown file. The community broadly loves the concept but is increasingly frustrated by gaps the spec's deliberate minimalism leaves unaddressed.

### Adoption

33+ tools including: Claude Code, OpenAI Codex, Gemini CLI, Cursor, GitHub Copilot, VS Code, Goose, Roo Code, Kiro, Spring AI, Databricks, Snowflake, JetBrains Junie, Mistral Vibe, Laravel Boost, OpenHands, Amp, Factory, Letta, Firebender, OpenCode, Mux, TRAE, Qodo, Emdash, Piebald, Command Code, Ona, VT Code, and more.

### Bottom Line

The spec won the adoption race through radical simplicity and is now paying the price for everything that simplicity left out. The community remains engaged (not abandoning) because the core concept is sound, but frustration is growing around governance responsiveness, security gaps, and Anthropic not following its own standard.
