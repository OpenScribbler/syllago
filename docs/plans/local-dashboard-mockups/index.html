Syllago Dashboard · Mockups

Four aesthetic variations of the local dashboard, each rendering the same mock catalog with the same search + filter behavior. All share the Flexoki palette and support light/dark mode — the difference is typography, density, shape, and tone. Open each, type in the search, click the chips, flip the theme.

red orange yellow green cyan blue purple magenta

Shared across all mockups: shared/tokens.css (Flexoki palette), shared/data.js (mock catalog), shared/filter.js (filter predicate — identical semantics everywhere), and shared/render.js (DOM helpers). No build step, no dependencies — open the HTML directly.

Keyboard: / focuses search on Terminal & Cyberpunk. ⌘K / Ctrl-K focuses search on Material & Tech Docs. Esc clears the search on any mockup.

Theme toggle persists per-browser via localStorage, and respects prefers-color-scheme as the default when no override is set.
