# Concise Output (Cursor)

A rule that instructs Cursor to keep responses focused and avoid unnecessary verbosity.

## What it does

Sets communication guidelines for Cursor's AI assistant:
- Avoids verbose preambles and summaries
- Keeps responses focused on the task
- Only provides detail when explaining trade-offs or when explicitly asked

## When to use

Install this rule when you want Cursor to be more direct and skip filler text.
